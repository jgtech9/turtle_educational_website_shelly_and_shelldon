# Images Directory

This directory contains all images for the Turtle World website.

## Directory Structure

- `header/` - Header logo and navigation images
- `pages/` - Main content page images (habitat, diet, species, etc.)
- `media/` - Media gallery images
- `crafts/` - Craft project images

## Required Images

1. **Header Icon** (`header/turtle-icon.png` or `.svg`)

   - `habitat-main.jpg` - Turtle habitats illustration

## Image Specifications

- **Main page images**: 600x400px (generate at 1200x800px)
- **Gallery images**: 300x200px (generate at 600x400px)
- **Icons**: 50x50px (generate at 200x200px)
- **Format**: JPG for photos, PNG for illustrations with transparency, SVG for icons

## Where to Get Images

### Option 1: AI Image Generation
Use the prompts in `IMAGE_GENERATION_PROMPTS.md` with AI image generators:
- DALL-E
- Midjourney
- Stable Diffusion
- Adobe Firefly

### Option 2: Stock Photos
Use royalty-free image sources:
- Unsplash (unsplash.com)
- Pexels (pexels.com)
- Pixabay (pixabay.com)
- FreeImages (freeimages.com)

Search terms: "turtle", "sea turtle", "tortoise", "turtle habitat", etc.

### Option 3: Commission Artwork
Work with an illustrator for custom educational illustrations.

### Option 4: Your Own Photos
If you have photos of Shelly and Shelldon, use those for the Shelley & Shelldon page.

## Temporary Placeholders

Until images are added, the code will use SVG placeholders. These work but should be replaced with actual images for the best user experience.

## File Naming Convention

- Use kebab-case: `turtle-species-comparison.jpg`
- Include page identifier: `habitat-sea-turtle.jpg`
- Be descriptive but concise
