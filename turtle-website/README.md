# Turtle Conservation & Fundraising Resources

This directory contains resources and data for displaying and linking to legitimate turtle conservation and fundraising organizations.

## Purpose

This directory serves to:
- Organize information about legitimate turtle conservation organizations
- Provide resources for fundraising and donation links
- Maintain a centralized list of verified conservation organizations
- Support the display of conservation resources across the website

## Structure

- `conservation-organizations.json` - Data file with legitimate conservation organizations
- `fundraising-organizations.json` - Data file with legitimate fundraising/donation organizations
- `README.md` - This file

## Usage

This data can be used to:
- Display conservation links on the Help page
- Create a dedicated conservation resources page
- Power a conservation organizations component
- Maintain up-to-date links to legitimate organizations

## Organization Criteria

Organizations included should be:
- Legitimate, verified conservation organizations
- Registered non-profits or recognized conservation groups
- Organizations with proven track records in turtle conservation
- Transparent about their work and use of funds
- Appropriate for educational use
